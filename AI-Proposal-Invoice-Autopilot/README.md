# 🚀 AI Proposal & Invoice Autopilot

A massive, high-reliability automation for consultative sales businesses. It automates the entire lifecycle from lead intake to professional PDF generation and follow-up.

## 🌟 Key Features
- **Dynamic Proposals**: Generates tailored Google Slides proposals based on form inputs.
- **Professional Invoices**: Simultaneously generates matching professional invoices.
- **Smart Follow-Up**: A multi-day (3, 7, 14 day) logic chain that checks the CRM before sending reminders—it "knows" if the client already paid.
- **Closing Logic**: Handles "Accept" and "Decline" webhooks to finalize the sales cycle.

## 🏗️ Technical Highlights
- **Fail-Safe Follow-Ups**: Uses centralized CRM lookups to prevent "hallucinated" follow-ups after a signature.
- **Template-Based**: Uses Google Slides and Docs as templates for pixel-perfect brand consistency.
- **Multi-Trigger**: Supports manual triggers, scheduled checks, and form submissions.

## ⚙️ Setup
- Requires Google Slides & Google Docs templates.
- Requires a structured Google Sheet or CRM to track proposal states.
