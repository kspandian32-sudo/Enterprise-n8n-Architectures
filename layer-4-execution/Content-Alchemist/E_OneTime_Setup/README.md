# Content Alchemist: ONE-TIME SETUP

## 📋 Role
A bootstrap utility that creates the entire Google Drive folder hierarchy and the primary Google Sheet database.

## 🔄 Flow
**Manual Trigger** -> **Create Root Folder** -> **Create Sub-Folders (7x)** -> **Create Google Sheet** -> **Initialize Tabs** -> **Output IDs**

## 💡 Why this way?
- **Zero-Config Deployment**: Removes the 15-minute manual task of creating folders and sheets.
- **Consistency**: Ensures that all sub-workflows find the exact folder structure they expect.
