# 📱 WhatsApp Business AI Bot Series

A collection of production-ready WhatsApp bots tailored for specific service-based businesses.

## 🏨 Hotel AI Concierge
- **Features**: Handles room availability inquiries, breakfast timing questions, and local recommendation requests.
- **Logic**: Uses a specialized "Hotel Knowledge Base" to provide accurate answers about the specific property.

## 🍽️ Restaurant AI Booking Bot
- **Features**: Automates table reservations, menu inquiries, and dietary restriction checks.
- **Integration**: Direct connection to Google Calendar or a Reservation Sheet to mark bookings.

## 🔗 Simple Webhook Handler
- **Features**: A lightweight boilerplate for receiving and acknowledging WhatsApp messages. Perfect for building custom integrations.

## ⚙️ Setup
- Requires Meta WhatsApp Business API Cloud credentials.
- Requires n8n AI Agent nodes (for Concierge and Booking bots).
