# Workflow C: Twitter/X Branch

## 📋 Role
Generates and archives formatted Twitter threads with matching AI visuals.

## 🔄 Flow
**Receive Thread Data** -> **Build DALL-E Prompt** -> **Generate Image** -> **Save .txt Thread** -> **Log to Google Sheets**

## 💡 Why this way?
- **Atomic Threads**: Handles entire multi-part threads as single units, preserving the structure for easy copy-pasting or automated posting.
- **Visual Continuity**: Parallel to the LinkedIn branch but with X-specific formatting logic.
